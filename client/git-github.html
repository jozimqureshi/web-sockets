Git & GitHub Setup Guide

Essential commands for setting up Git, syncing with GitHub, and managing branches

Step
Command / Explanation
Initialize Git locally
git init
Set credentials globally (once)
git config --global user.name "jozimqureshi"
Set global email
git config --global user.email "jozimqureshi@gmail.com"
Check current branch name
git branch
Rename "master" to "main"
git branch -M main
Check remote name & URL
git remote -v
Pull with unrelated histories
git pull origin main --allow-unrelated-histories
What it does
Downloads only changed files from GitHub
Avoid repeating branch name
git pull -u origin main
Download remote changes
git pull
Upload local changes # Stage all changes
git add .
Commit changes # Commit them locally
git commit -m "Add full local project"
Push to GitHub # Push those commits to remote
git push
Check current Git status
git status
