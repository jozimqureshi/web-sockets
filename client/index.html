🧠 WebSocket Lifecycle Demo

This demo illustrates how a WebSocket connection works between a client and server,
using a real-life shopping mall scenario where a customer interacts with store staff.

🧠 WebSocket: Full Lifecycle with Technical and Real-Life Analogies

🔢 Step
📜 WebSocket Code
💬 Technical Message
🏪 Shopping Analogy
1️⃣
new WebSocket(...)
Client is making a connection request to the server.
🔁 Attempting handshake.
🛒 Customer walks up to the mall entrance, trying to get in.
2️⃣
socket.onopen
✅ Client confirms the connection is open.
Client is now ready to send/receive messages.
🚪 The door opens and the customer steps into the store.
3️⃣
wss.on("connection")
✅ Server accepts connection from a new client.
The WebSocket handshake is successful.
👀 Staff sees a new customer walk in and greets them.
4️⃣
socket.send("Hello from Client")
📤 Client sends a message to the server.
Data is transmitted over WebSocket.
🙋‍♂️ Customer says: "Hello! Do you have this item?"
5️⃣
ws.on("message")
📥 Server receives the message from client.
Ready to process or respond.
👂 Staff hears the customer's question.
6️⃣
ws.send("Hello from Server")
📤 Server sends a reply to the client.
The response is on its way.
🧾 Staff replies: "Yes, it’s in aisle 3!"
7️⃣
socket.onmessage
📥 Client receives a message from the server.
Can now update UI or take action.
😊 Customer hears the answer and walks to aisle 3.
