  • First must initialze Git locally: git init
  • First set credentials one time globally:
  • git config --global user.name "jozimqureshi"
  • git config --global user.email "jozimqureshi@gmail.com"
  • Then check if local branch name is "Master" or "Main". It should match the github remote name: "Main". To do this, use: git bracnh
  • If found "Master", change the local branch name to "Main" using: git branch -M main
  • To check remote name (like origin) and the URLs: git remote -v
  • For different file content on both client and remote, use: git pull origin main --allow-unrelated-histories
  • This will download only changed files from github to git
  • To avoid repeating origin main use --set-upstream or simply -u for only 1 time: git pull -u origin main
  • To download all changed files from remote to local, use: git push
  • To upload all changed files from git to github, first add, commit and then push:
  • Here . means all: git add .
  • git commit -m "Any suitable commit remarks like: Add full local project"
  • git push
  • To check current status: git status
